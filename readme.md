# Microblog Service

Сервис микроблогов (аналог Twitter) для корпоративного использования.  
Разработан на **FastAPI** с использованием **PostgreSQL** в качестве базы данных.

## 🚀 Функциональность

- ✅ Создание и удаление твитов
- ✅ Загрузка изображений к твитам
- ✅ Подписка (follow) и отписка (unfollow) от пользователей
- ✅ Лайки и дизлайки твитов
- ✅ Лента твитов от подписок с сортировкой по популярности
- ✅ Профиль пользователя с подписчиками и подписками
- ✅ Авторизация через api-key (без регистрации)
- ✅ Swagger документация

## 🐳 Запуск через Docker

### Требования
- Установленный Docker Desktop
- 2 ГБ свободной оперативной памяти

### Быстрый старт

```bash
# 1. Перейти в папку с проектом
cd fastapi-microblog

# 2. Запустить контейнеры
docker-compose up --build -d

# 3. Создать тестовых пользователей
docker-compose exec web python seed.py

# 4. Проверить что всё работает
# Открыть в браузере: http://localhost:5000/docs
